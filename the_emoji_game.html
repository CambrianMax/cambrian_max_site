The Emoji Game

I am quite proud of this game. It uses my own C opengl engine. The windows version uses the Window's Api for creating windows, dsound for audio, and my own opengl loader to load in opengl function calls. I used SDL2 for the Linux and Mac ports. I started working on this game in September of 2023 and Finished it February of 2024.

You can download it on itch.io for FREE here!
